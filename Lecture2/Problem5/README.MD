## 实现思路

中缀表达式转换为后缀表达式的实现思路通过栈来实现。

具体来说，可以按照以下步骤实现中缀表达式转换为后缀表达式：

1. 初始化运算符栈，将#运算符压入栈中。
2. 将中缀表达式字符串的首尾加上()和#，作为待处理的字符串。
3. 遍历待处理字符串，对于每个字符，按照以下规则处理：
    - 如果是数字，则直接加入后缀表达式字符串中。
    - 如果是左括号，则将其压入运算符栈中。
    - 如果是右括号，则将栈中元素弹出并加入后缀表达式字符串中，直到遇到左括号为止。左括号弹出后直接丢弃。
    - 如果是运算符，则将其与栈顶运算符进行比较。
        - 如果栈顶运算符优先级更高，则将栈中运算符弹出并加入后缀表达式字符串中，直到栈顶运算符优先级小于该运算符。
        - 最后将该运算符压入栈中。
4. 遍历完待处理字符串后，将运算符栈中剩余的元素依次弹出并加入后缀表达式字符串中。
